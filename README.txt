To Run Bid Program (bidder_app) 

1. In your terminal, navigate to the project directory

2. Install Flask modules 
	$ pip install -r requirements.txt

3. To Run unit tests
	$ python test_bidder.py

4. To Start the Flask Application:
	$ python run.py



Note: 
All search posts should be sent to http://localhost:5000/
Program requires python 3.5 or later to be installed
Program only tested on Windows operating system.
You can view UI friendly records in the database by using: http://sqlitebrowser.org/
